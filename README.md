Project Portofolio Iin Sumarni

Proses pembuatan website portofolio ini saya lakukan secara bertahap melalui beberapa commit di Git. Tahap pertama saya mulai dengan membuat project dasar sebagai pondasi utama dari website. Setelah struktur awal terbentuk, saya menambahkan elemen-elemen penting seperti header, navbar, dan landing page agar tampilan utama situs bisa terbentuk dengan baik. Setelah itu, saya menambahkan bagian portofolio dan pengalaman organisasi untuk melengkapi isi website serta menampilkan informasi pribadi secara lebih lengkap.

Pada tahap berikutnya, saya fokus pada pengaturan tampilan. Di tahap ini, saya menambahkan footer dan menerapkan style dasar menggunakan CSS supaya tampilan halaman terlihat lebih rapi dan konsisten. Setiap bagian seperti navbar, landing page, foto, portofolio, dan organisasi juga saya beri gaya tersendiri agar tampilannya lebih menarik dan tidak monoton. Setelah itu, saya menambahkan animasi serta mengatur tampilan agar responsif, sehingga website tetap terlihat baik di berbagai ukuran layar. Saya juga menambahkan semua aset gambar untuk melengkapi elemen visual pada halaman.

Sebagai tahap akhir, saya melakukan penyesuaian warna supaya tampilan website lebih serasi dan sesuai dengan tema yang saya inginkan.
